# My Project FlaskApp

Simple application with authentication using the Python Flask micro-framework

## Installation

To use this template, your computer needs:

- Install requerements.txt
- Install Mysql<=8
- Install docker (optional)
- Install docker-compose (optional)


### Running the app

```bash
python app.py
```
